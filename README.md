# Лабораторная работа №4. Unit-тесты для функций нахождения площади и периметра круга

## Функции

- `area(r)` — вычисляет площадь круга по формуле \( S = \pi r^2 \)
- `perimeter(r)` — вычисляет периметр (длину окружности) по формуле \( P = 2\pi r \)

## Тесты

Тесты реализованы в файле `test_circle.py` с использованием стандартного модуля `unittest`  

## Как запустить тесты

В терминале выполните команду:

```bash
python -m unittest test_circle.py -v
